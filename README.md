# Omega Digital

A creative web design & development studio website built with React, TypeScript, Tailwind CSS, and Vite.

## Getting Started

```bash
npm install
npm run dev
```

## Project Structure

```

├── src/
```

## Tech Stack

- **React 19** — UI framework
- **TypeScript** — Type safety
- **Tailwind CSS 3** — Utility-first styling
- **Vite 7** — Build tool
- **Lucide React** — Icons
